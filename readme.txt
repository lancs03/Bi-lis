一、文件usb_Driver software.exe是USB的驱动软件。

二、启动程序Bi-LIS.exe，用串口线连接A5000.（如电脑不识别串口线，要先安装驱动）。

三、Parameter settings->Ports settings->Refresh,勾选Bi-LIS test选项，然后Save changes。
（注意：在Parameter settings->System settings中选项device要和A5000的系统名称一致）

四、Parameter settings->Bi-LIS test，录入试管条码和要做对应项目，就可以做双向lis测试。

五、在Report entry中可以看到传回的测试结果。

六、在文件夹bi-lbs中记录串口在传输过程中的数据。

七、在线升级dev+mas
